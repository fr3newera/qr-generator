SDK v1.0

FR3QRIS SDK

Integrasikan QRIS Generator FR3 NEWERA ke website manapun hanya dengan beberapa baris kode. Mendukung embed iframe, download otomatis, dan pengambilan data URL gambar.

Instalasi

<!-- Tambahkan tag script ini di website Anda -->
<script src="https://qris-generator.fr3newera.com/qris-sdk.js"></script>

Referensi API

function FR3QRIS.generate(options) Embed QR ke elemen HTML 
const qr = FR3QRIS.generate({
  qrcode:  "00020101...",        // String QRIS / URL
  name:    "Nama Toko Anda",   // Nama merchant
  target:  "#qris-container",  // CSS selector atau element
  width:   "320px",            // Lebar iframe (opsional, default 100%)
  onReady: function(res) {   // Callback saat selesai render
    console.log(res.dataUrl);  // Data URL gambar PNG
  }
});

qr.download();  // Download gambar QRIS
qr.destroy();  // Hapus iframe dari DOM
function FR3QRIS.download(options) Download langsung tanpa embed 
FR3QRIS.download({
  qrcode: "00020101...",
  name:   "Nama Toko",
  onDone: function() { console.log("Selesai!"); }
});
promise FR3QRIS.getDataUrl(options) Ambil data URL gambar (Promise) 
FR3QRIS.getDataUrl({ qrcode: "...", name: "..." })
  .then(function(dataUrl) {
    const img = document.createElement("img");
    img.src = dataUrl;
    document.body.appendChild(img);
  })
  .catch(function(err) { console.error(err); });
iframe Embed langsung via URL Tanpa SDK, pakai iframe manual 
<iframe
  src="https://qris-generator.fr3newera.com/?embed=1
       &qrcode=00020101...
       &name=Nama+Toko"
  width="320"
  style="border:none; aspect-ratio:1/1;"
></iframe>
